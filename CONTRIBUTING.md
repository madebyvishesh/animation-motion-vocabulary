# Contributing

Contributions are welcome.

## Good Changes

- Add missing animation vocabulary terms.
- Improve near-match phrase mappings.
- Add clearer examples for ambiguous prompts.
- Improve install instructions for agents that support `SKILL.md`.
- Fix wording that could make an agent over-animate an interface.

## Editing Guidelines

- Keep `animation-motion-vocabulary/SKILL.md` concise.
- Put detailed glossary entries in `animation-motion-vocabulary/references/animation-motion-vocabulary.md`.
- Keep the standalone `animation-motion-vocabulary.md` in sync with glossary changes.
- Avoid adding scripts unless the skill genuinely needs automation.
- If scripts are added later, document what they do and why they are safe.

## Pull Request Checklist

- The skill folder still contains `SKILL.md`.
- New terms include plain-language meanings.
- Ambiguous terms include guidance on when to confirm with the user.
- Accessibility and reduced-motion guidance remains intact.
